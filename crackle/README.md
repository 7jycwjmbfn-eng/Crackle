# Crackle Package

This package contains the code for the Crackle fracture-AI research archive.

Main modules:

- `crackle.baselines`: deterministic, Hawkes-style, survival, and GBM baselines.
- `crackle.data`: causal event catalogs, risk sets, and feature builders.
- `crackle.eval`: CrackMNIST, DLR DIC, NASA curve, wall-clock, and head-to-head evaluators.
- `crackle.experiments`: controlled heterogeneous pinning / rollout-cut experiments.
- `crackle.metrics`: fracture and point-process metrics.
- `crackle.train`: training entry points.

See the repository-level `README.md` for installation, data sources, example commands, and claim boundaries.
